# 🤝 Code of Conduct

## 1. My Commitment

I am committed to making this project a welcoming, inclusive, and respectful space for everyone.

This project is built for players, contributors, and learners of all experience levels.  
I value collaboration, learning, and constructive improvement.

---

## 2. Expected Behavior

All participants are expected to:

- Be respectful and considerate in all interactions
- Provide constructive and helpful feedback
- Focus on improving the project, not attacking individuals
- Respect differing opinions and approaches
- Acknowledge that everyone is learning and improving

---

## 3. Unacceptable Behavior

The following behaviors are not allowed:

- Harassment, discrimination, or offensive language of any kind
- Personal attacks or insulting comments
- Spamming, trolling, or disruptive behavior
- Publishing private or sensitive information without permission
- Any behavior that creates an unwelcoming environment

---

## 4. Scope

This Code of Conduct applies to all project spaces, including:

- GitHub Issues
- Pull Requests
- Discussions and comments
- Any future community channels related to this project

---

## 5. Enforcement

Maintainers have the right to:

- Remove or edit comments or contributions that violate this Code of Conduct
- Reject or close pull requests that do not meet community standards
- Temporarily or permanently restrict access to the project if necessary

All reports of unacceptable behavior will be reviewed fairly and respectfully.

---

## 6. Reporting Issues

If you experience or witness behavior that violates this Code of Conduct, please report it by:

- Opening a GitHub Issue
- Or contacting the project maintainer directly

All reports will be handled confidentially and seriously.

---

## 7. Attribution

This Code of Conduct is inspired by the Contributor Covenant (version 2.1) and adapted for this project.

---

## ⭐ Final Note

This project is built on collaboration, learning, and respect. By participating, you agree to uphold these principles.
